IT 
インターネットなどの通信とコンピュータとを駆使する情報技術。

OS
オペレーティングシステムの略称。スマホやパソコンの基本動作を司る脳みそ
Windows
microsoftがOSを開発している。世界的中で安定して利用されており、関連製品やデザインが豊富
macOS 　
appleがOSを開発している。
UNIX　
現存する中で最も古いOS、長時間にわたって動かす必要のあるコンピュータなどに利用される
Linux　
Unixを参考につくられたUNIX系OS、パソコンやスマートフォン、多くのサーバーとして利用される。

HTML　
静的なWebサイトを制作するのに必要。文字、画像の表示や文章を構成するための言語。家でいう骨組みの部分。
CSS　
静的なWebサイトを制作するのに必要。文字の大きさや色、配置を変更や背景を設定する言語
MySQL　
データベース（複数で共有、利用することと、検索、加工することを目的に整理されたデータの集まり）管理システム。
PHP　
Webサイトに動的な機能を実装すること。サーバーサイド言語であり、処理はWebサーバーでおこなわれる。
JavaScript　
Webサイトに動的な機能を実装すること。クライアントサイド言語とよばれており、ユーザー側のブラウザで処理がおこなわれる。
jQuery
JavaScriptでできることを、より簡単な記法で実現できように設計されたJavaScriptライブラリ
Apache
世界的に最も普及しているWebサーバ（HTTPサーバ）ソフトウェアの一つ
Bootstrap
Web制作者の作業を楽にする目的で作られた、Webサイト構築用のCSSひな形セット
一般的にコンピュータシステムの電源を入れてからOSを起動して利用可能な状態になるまで自動的に実行される処理を指す。

SE
どんなシステムにするか設計するのが仕事の人
PG
実際のプログラムを作るのが仕事の人
Webデザイナー
ホームページの見た目に関する部分を考えたり作ったりする人のこと

CUI
すべての操作をキーボードだけで操作し動作をコンピュータに指示すること
GUI
マウスや指で操作しあらかじめセットされたプログラムが作動すること

ファイル
コンピュータにとって分かりやすい形でまとめたデータのこと。拡張子がついているもの
ディレクトリ
ファイルを整理・保管するための袋（ディレクトリはフォルダともいう。）
ルート・ディレクトリ
階層の一番上にあるディレクトリ。
カレント・ディレクトリ
現在 作業しているディレクトリ。

絶対パス
URLでページを指定して、目的地（情報）がどこにあるのかを確実に伝える。
相対パス
今いる場所（階層）を基準にして、目的地（情報）がどこにあるのかを伝える。

HTTP　
WebサーバとWebクライアントの間でebページを構成するHTMLなどのデータの送受信を行うために用いられるプロトコル（通信規約）
FTP
特定のコンピュータ間でファイルを送受信する時に使う決まりごと（通信プロトコル）
HTTPS
WebサーバとWebクライアントの間でebページを構成するHTMLなどのデータの送受信を行うために用いられるプロトコル（通信規約）httpと違い通信内容が暗号化されている。
SSL
インターネット上の通信を暗号化する仕組み
SSH
他のコンピュータを遠隔操作する際のやり取りを暗号化する仕組み
ポート、ポート番号
TCP/IP通信において、 コンピュータが通信に使用するプログラムを識別するための番号。IPアドレスを住所とするならばポート番号は部屋番号のようなもの
HTTP、FTP、SSHの各ポート番号
HTTP…80番
FTP…20番（サーバー側からクライアント側にデータコネクションの接続要求時）、21番（クライアント側からサーバー側にコントロールコネクションの接続要求時）。
SSH…22番
ファイアウォール
ネットワークの境界に設置され、内外の通信を中継・監視し、外部の攻撃から内部を保護するためのソフトウェアや機器のこと。
TCP
通信相手の状況を確認して接続を確立し、データの伝送が終わると切断するコネクション型のプロトコルのこと。

IPアドレス
インターネットをするときにコンピュータに割り当てられる住所
グローバルIPアドレス
全世界で通用するネットワーク上の住所
プライベートIPアドレス
仲間内でしか通用しないネットワーク上の住所

ドメイン
範囲、領域などの意味。ITの分野ではインターネットなどのTCP/IPネットワーク上で機器やネットワークを識別するドメイン名（domain name）を指す
DNS
インターネットなどのIPネットワーク上でドメイン名（ホスト名）とIPアドレスの対応関係を管理するシステム
VPN
通信事業者の公衆回線を経由して構築された仮想的な組織内ネットワーク
ブリッジ
プロキシサーバー
インターネットに直接接続できないコンピューターに代わり、インターネットに接続し、Webサイトへのアクセスなどを行うサーバーのこと
電子メール
コンピュータ通信ネットワーク上で、文書・画像などの情報を伝達するための通信システム。
SMTP
メールを送信する仕組み。メールを相手のメールサーバーまで届ける
POP
メールを受信する仕組み。メールソフトが自動的にメールサーバーにアクセスしてメールをダウンロードしている仕組み。メールをいったんスマホやパソコンにダウンロードして取り込むと、サーバーからは無くなってしまう。
IMAP
メールを取り込むための手順。popと違いメールをサーバーに残しているもの

セッション
コンピュータシステムやネットワーク通信において、接続/ログインしてから、切断/ログオフするまでの、一連の操作や通信のこと
クッキー
Webサイトの提供者が、Webブラウザを通じて訪問者のコンピュータに一時的にデータを書き込んで保存させる仕組み
キャッシュ
訪問した “ ウェブページの情報 ” を一時的に保存する仕組み

チューニング　
システムやプログラムなどを改良し、パフォーマンスの向上を図ること。
スロークエリ
データベースにおいて、時間のかかっているSQL(遅いSQL)のこと
インデックス
個々の要素を区別するために付けられた通し番号などの識別情報

フロー
コンピュータプログラムを構成する命令や手続き、処理の流れなどのこと
単体テスト
合体する前のプログラムいわば部品がきちんと動くか確認するテストのこと
結合テスト
合体したプログラムが正常に動くか確認するテストこと

チケット管理ツール
実施するべき作業や課題を「チケット」として管理する方法をチケット管理といい、そのツールのこと。
タスク
コンピューターが処理する作業の最小単位のこと

Git（分散型バージョン管理システム）
プログラムソースなどの変更履歴を管理するシステムのこと。コミット単位に変更履歴を管理。
SourceTree
Gitの操作ができる無料のデスクトップアプリケーション
SVN（集中型バージョン管理システム）
ファイルの変更履歴を管理してくれるシステムのこと。ファイルごとに変更履歴を管理。
コンフリクト
複数の装置やプログラムなどが同じ資源を同時に利用しようとして競合状態になってしまうことを意味する場合
デグレ（デグレード）
新しいバージョンのソフトウェアの品質が、以前より悪くなること。
マージツール
2つ以上のデータを一つのデータにまとめるコンピュータ処理のことをマージといい、そのツール（道具）のこと。
